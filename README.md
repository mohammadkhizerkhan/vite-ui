# vite-ui
 
## Description-
Vite-UI is a CSS library for developing quick and clean reponsive components just by copy pasting.
<hr>

## Technologies 
- HTML
- CSS
- JS
<hr>

## Components defined in Vite-UI:
- Alert
- Avatar
- Badges
- Button
- Cards
- Navigation
- Image
- Input
- Modal
- Text-Utilities

<hr>
